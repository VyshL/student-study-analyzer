# Student Study Analyzer

Student Study Analyzer is a web application that helps students understand how their study time and sleep habits affect academic performance.

Users can dynamically enter the number of subjects, subject names, marks, study hours, and sleep hours. The application generates real-time visualizations and provides basic insights to support better academic habits.

---

## Features
- User-defined number of subjects (up to 8)
- Custom subject names
- Interactive sliders for marks, study hours, and sleep hours
- Dynamic charts generated using Matplotlib
- Insight messages based on user input

---

## Tech Stack
- Python
- Streamlit
- Matplotlib

---

## Purpose
This project was built to solve a real-life problem by helping students analyze their academic habits through interactive data visualization.
